# HealthExamination

## 简单的学校疫情上报系统自动化方法

#### 一、制作原因

解决忘报问题

#### 二、使用方法

1.自行抓包

需要 `setid`、`userid`、`id`、`Auth-Token`

修改配置文件`config.json`，替换#为数值

2.运行 `python3 main.py`

windows用户 使用 PowerShell，cd对应目录运行命令就可以

3.需要通知可在`config.json`添加`ServerKey`

使用[Server酱Turbo](https://sct.ftqq.com/)通知
